# Aceapp (Education Web App)

A static HTML/CSS/JS education platform for students with:

- Improved authentication flow (signup/login/session)
- Professional student dashboard with summary cards
- Admin panel to add tests, questions, PDFs, and video links
- Free/Paid content access rules
- Payment unlock flow (demo unlock)
- Free PDF download, paid PDF read-only flow
- YouTube lecture support

## Default admin credentials
- Mobile: `9999999999`
- Password: `admin123`

## Storage
The app uses browser `localStorage` for users, session, content and unlocked purchases.
